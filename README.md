# 🛋️ Dungeon Stroll
A mobile game using user geolocation to play

## Overview  
Its a mobile game in which player controls group of characters exploring a dungeon. Player can enter dungeon by getting to its geolocation and interacting with it.
Once inside several points of interest appear, those can be either fights or treasure. The combat takes place in an autobattler style where once it starts player has no impact on it.
Once all points of interest are visited or all heroes defeated dungeon exploration ends and its result is recorded.
Using the gold collected in dungeon player may upgrade heroes and their abilities.
Player may also see history of his previous attempts of fighting in the dungeons
